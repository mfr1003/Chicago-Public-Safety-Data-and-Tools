## Request

Pursuant to the Illinois Freedom of Information Act (5 ILCS 140), I respectfully
request the following records related to the Chicago Police Department's transit
section operations and fare enforcement activities.

### Record Set 1: Transit Section Budget

For each calendar year 2018 through present, records sufficient to show:

1. Total allocated budget for the CPD Transit Section (or equivalent unit
   responsible for CTA patrol and enforcement)
2. Actual expenditures for the CPD Transit Section by year
3. Headcount (sworn officers and civilian staff) assigned to the Transit Section
   by year, including full-time equivalents if applicable
4. Any portion of the Transit Section budget funded by or reimbursed by the
   Chicago Transit Authority

### Record Set 2: Activity Code Breakdown

For each calendar year 2018 through present, records sufficient to show officer
hours or activity logs broken out by activity type, specifically:

1. Hours logged to fare enforcement activity (farecard inspection, fare evasion
   citation issuance, or equivalent activity code)
2. Hours logged to general patrol on CTA property
3. Hours logged to crime response on CTA property (assault, robbery, battery,
   homicide, or equivalent codes)
4. Any other activity code categories used by the Transit Section

If CPD does not track activity hours by the specific categories listed above,
please provide whatever activity code or deployment log records exist for the
Transit Section, along with documentation of the coding system used.

### Record Set 3: Citation Issuance

For each calendar year 2018 through present:

1. Total number of ANOVs issued by Transit Section officers for transit-related
   violations, broken out by violation code
2. Total number of arrests made on CTA property by Transit Section officers,
   broken out by charge category
